# CONTADOR de CLICKS

Un contador de clicks es algo muy sencillo de entender, no tiene la gran lógica, de hecho ya con tan solo leer "Contador de clicks" sabemos de que estamos hablando...  Pero si vamos al caso es un acción que nosotros hacemos, la cual la podemos hacer clickeando un mouse o también pulsando si hablamos de un dispositivo mobil,  seguido a eso se "respondera" con la cantidad de veces que hemos repetido la misma.

## Tutorial

`Vamos a empezar creando 3 archivos:`

> - Archivo `index.html`
> - Archivo `style.css`
> - Archivo `main.js`

Esos fueron los nombres que yo utilize, ya ustedes después pueden elegir el nombre que deseen.

> Dentro del archivo **HTML** (`index.html` en mi caso) armaremos obviamente la estructura de la página, para especificar mejor voy a dejar el código a continuación.

**HTML**

```html
<!--  Vinculamos el archvio JavaScript -->
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
     <!--  Vinculamos el archvio CSS -->
    <link rel="stylesheet" href="style.css">
    <title>Contador de clicks</title>
</head>
<body>
    <!--  Creamos un DIV el cual va a contener todo -->
    <div class="content">
        <!--  Creamos un BOTÓN al cual le agregaremos un evento -->
        <button class="button">Click</button>
        <div class="counter">0</div>
    </div>
</body>
<!--  Vinculamos el archvio JavaScript -->
<script src="main.js"></script>
</html>
```

> Seguido de la estructura, vamos a darle unos estilos. Yo elegi dar estos estilos, cada uno puede hacerlo como quiera si es que es dinamico con **CSS** o de lo contrario pueden usar estos mismos!

**CSS**

```css
* {
    padding: 0;
    margin: 0;
    box-sizing: border-box;
    font-family: sans-serif;
}

body {
    display: flex;
    justify-content: center;
    align-items: center;
    margin-top: 200px;
    background: #3c3c3c;
}

.content {
    padding: 15px 25px;
    border: 1px solid rgba(255, 255, 255, .2);
    border-radius: 10px;
}

.button {
    padding: 15px 25px;
    border: 2px solid green;
    border-bottom: 5px solid green;
    border-radius: 10px;
    background: rgba(255, 255, 255, .2);
    font-size: 20px;
    color: #fff;
    cursor: pointer;
    transition: border-bottom .1s ease-in-out;
}

.button:active {
    border-bottom: 2px solid green;
}

.counter {
    font-size: 25px;
    color: #fff;
    text-align: center;
    padding: 10px 20px;
    margin-top: 10px;
    border: 1px solid green;
    border-radius: 5px;
    user-select: none;
}
```

> Y por último lo que hara que todo funcione, es nuestro archivo **JavaScript**

**JS**

```js
const button = document.querySelector('.button');
const counter = document.querySelector('.counter');
let count = 0;

button.addEventListener('click', () => {
    count++;
    counter.innerText = count;
    counter.style.backgroundColor = 'green'
})
```
